Power Source-Based Switching
This feature is disabled by default, and does not currently work with MBPs that are not 2010 models. If you wish to use it, simply enable it in gfxCardStatus preferences and explicitly set which mode you wish to use on a per-power source basis with the segmented controls. If you do not enable it, gfxCardStatus will behave exactly as before.

A word of caution: this feature is young and, and as such should be considered a beta feature. Please contact me if any issues arise at the email address listed above.
